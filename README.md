# CPR模拟训练系统

## 项目简介
CPR模拟训练系统是一个基于Phaser 3游戏引擎开发的交互式心肺复苏训练平台。该系统通过模拟真实的CPR操作场景，帮助用户学习和掌握心肺复苏的正确操作流程和技巧。系统采用现代化的Web技术栈，提供直观的用户界面和实时的操作反馈。

## 技术栈
- 前端框架：Phaser 3
- 开发语言：JavaScript
- 构建工具：Webpack
- 包管理器：npm
- 代码规范：ESLint

## 功能特点
- 完整的CPR操作流程模拟
- 实时操作反馈和指导
- 交互式训练场景
- 详细的错误提示和纠正
- 进度追踪和统计
- 多场景切换
- 音效和动画效果

## 使用说明
### 训练流程
1. 场景介绍
2. 检查意识
3. 呼叫救援
4. 开放气道
5. 胸外按压（30次）
6. 人工呼吸（2次）
7. 循环操作
8. 训练完成

### 操作指南
- 胸外按压：
  - 按压时间：100-300毫秒
  - 按压深度：5-6厘米
  - 按压频率：100-120次/分钟
  
- 人工呼吸：
  - 呼吸时间：1000-2000毫秒
  - 呼吸次数：2次
  - 按压呼吸比例：30:2
